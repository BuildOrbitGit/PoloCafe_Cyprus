# Caffe Polo Venezia Limassol - Master Website Prompt

Use this prompt to recreate or continue the current Caffe Polo Venezia website direction. The goal is a premium, mysterious, cinematic cafe website with a red and black identity, minimal homepage text, elegant motion, and strong use of the provided coffee media.

## Brand

Brand name: Caffe Polo Venezia

Location: Limassol, Cyprus

Instagram: https://www.instagram.com/polocafelimassol/

Mood words: mysterious, premium, cinematic, elegant, dramatic, refined, quiet luxury, red smoke, black atmosphere, Italian cafe energy, high-end coffee experience.

Primary palette:
- Deep black: `#050505`
- Espresso black/brown: `#1a0f0b`
- Signature red: `#c9002b`
- Dark red: `#87001e`
- Warm cream: `#f4eee7`
- Muted coffee beige: `#c8b9ad`

Typography direction:
- Use a strong serif display font for the brand name and large headings.
- Use clean uppercase sans-serif text for buttons, labels, and navigation.
- Keep letter spacing controlled. Do not over-stretch text.

## Master Build Prompt

Build a premium one-page website for Caffe Polo Venezia in Limassol. The homepage must feel mysterious and cinematic, not like a normal cafe landing page. Remove any traditional hero video, old header design, and logo usage from the homepage. The first screen should be simple, dramatic, and memorable.

The homepage background must be a full-screen animated red smoke / liquid shader effect on black. The smoke should flow smoothly from right to left with visible movement, giving the user an immediate feeling of mystery. The animation should be faster than a subtle ambient background so users can clearly see the red smoke evolving. The smoke should be strong near the edges and corners, but the center area around the main text should remain visually calmer, like an imaginary circular quiet zone. Do not draw an actual circle. Create the feeling of a protected center through opacity, gradients, vignette, and smoke density control.

In the exact center of the homepage, place only the cafe name:

`Caffe Polo Venezia`

Make the name very large, white, centered, and elegant. It should dominate the first screen. Under the name, place exactly two options:

`Menu`

`Location`

These two options should be white filled rectangular buttons with black uppercase text. Keep them refined and minimal. Do not add extra subtitles, descriptions, hero copy, logo, or decorative text on the homepage.

Add a fixed header overlay that remains visible as the user scrolls. On the top-left, place a large Instagram icon inside a white circular button. Link it to:

`https://www.instagram.com/polocafelimassol/`

The Instagram button should be attention-grabbing but clean. In the top center of the fixed header, show `Caffe Polo Venezia` in white serif text. This centered header text should stay visible as the user scrolls down the site.

Add a homepage background photo effect using four selected cafe images. The images should appear in a white polaroid-style frame. Only one polaroid should be visible at a time. Every 2 seconds, fade out the current polaroid and fade in a new one. The photo should randomly appear in one of the four corners: top-left, top-right, bottom-left, bottom-right. The movement should feel casual, subtle, and editorial, like memories appearing around the smoke. The polaroids should be big enough to capture attention, but they must remain corner photos and never cover the main cafe name or buttons. Use small rotations for a natural polaroid feel.

Current intended polaroid sizing:
- Desktop: `width: clamp(240px, 18vw, 330px)`
- Mobile: `width: clamp(132px, 34vw, 168px)`

Keep the homepage layout uncluttered. The hierarchy should be:
1. Red smoke background
2. Corner polaroid moment
3. Huge white cafe name
4. Menu and Location buttons
5. Fixed Instagram/header overlay

## Section Structure

The site should remain a focused one-page experience:

1. Homepage / Hero
   - Red smoke shader background.
   - Huge centered brand name.
   - Menu and Location buttons.
   - Fixed Instagram icon top-left.
   - Fixed cafe name top-center.
   - Random corner polaroid photo animation.

2. Gallery
   - Use the provided coffee photos and videos.
   - Keep the design editorial and premium.
   - Avoid generic stock-style layouts.
   - Images should feel warm, close-up, sensory, and cafe-focused.

3. Menu
   - Dark red/black premium menu section.
   - Use a subtle coffee video or espresso detail background with heavy dark overlay.
   - Menu categories can be tabbed.
   - Keep cards/panels restrained and professional.

4. Location / Visit
   - Include Limassol, Cyprus.
   - Include Instagram link.
   - Include map area or location panel.
   - Keep the dark cinematic palette consistent.

## Interaction And Motion Rules

- Smoke background should feel alive and flowing, not static.
- Polaroids should transition every 2 seconds.
- Polaroid transition should use fade + slight scale + slight vertical movement.
- Do not let polaroids cover the main title.
- Buttons should have clean hover states.
- Header should remain fixed across the site.
- Instagram button should stay accessible while scrolling.
- Use smooth scrolling between homepage buttons and sections.
- Avoid excessive animation outside the homepage; keep the rest premium and controlled.

## Design Rules

- Do not use the logo in the homepage hero.
- Do not use the previous traditional header in the homepage hero.
- Do not use a homepage hero video.
- Do not add marketing paragraphs to the homepage.
- Do not create a busy landing page.
- Keep the homepage mysterious and minimal.
- Use red and black as the dominant visual identity.
- Use white sparingly for contrast: title, buttons, polaroid paper, Instagram circle.
- Avoid rounded pill buttons for Menu and Location. Use clean rectangular filled boxes.
- Avoid beige or brown-heavy layouts. Coffee browns can appear in images, but the interface should remain red/black.
- Use real provided media. Do not replace the cafe imagery with unrelated stock photos.

## Technical Implementation Notes

Use Next.js with React components. Use a client-side smoke background component for the shader canvas. The homepage should render the smoke canvas full-screen behind all content.

Suggested components:

- `SmokeBackground`
  - Full-screen canvas.
  - Red smoke/liquid shader.
  - Right-to-left movement.
  - Stronger smoke near edges.
  - Calmer center around title.

- `PersistentHeader`
  - Fixed header.
  - Instagram link top-left.
  - Cafe name centered.

- `HeroPolaroids`
  - State-driven active image and active corner.
  - Change image and corner every 2 seconds.
  - Use fade/scale motion.
  - Render one polaroid at a time.

- `HeroSection`
  - Smoke background.
  - Polaroid layer.
  - Mystery overlay.
  - Centered title and two buttons.

Layering target:

- Smoke canvas: background
- Mystery overlay: above smoke
- Polaroids: above overlay but below main title
- Hero content: above polaroids
- Fixed header: top layer
